<p>준규가 가지고 있는 동전은 총 N종류이고, 각각의 동전을 매우 많이 가지고 있다.</p>

<p>둘째 줄부터 N개의 줄에 동전의 가치 A<sub>i</sub>가 오름차순으로 주어진다. (1 ≤ A<sub>i</sub> ≤ 1,000,000, A<sub>1</sub> = 1, i ≥ 2인 경우에 A<sub>i</sub>는 A<sub>i-1</sub>의 배수)</p>

#### 아쉬운 점

- K < c인 경우를 continue로 따로 처리하도록 했으나, 정수 나눗셈 특성상 K // c가 0이므로 분기 없이도 동일하게 동작했다.
- 마찬가지로 c == 1 예외 처리도 필요 없었다. 동일 로직(cnt += K // c, K %= c)으로 처리하면 자동으로 K가 0이 된다.
- 불필요한 if/continue 분기를 제거하고 한 줄(cnt += K//c; K%=c)로 단순화하면 더 좋았겠다.
  
```python
for c in coins:
    cnt += K // c
    K %= c
```
