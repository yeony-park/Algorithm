### 오답 유형

| 오답 | 유형 | 설명 |
|------|------|------|
| `RE` | Runtime Error | 실행 중 예외 발생 (IndexError, ZeroDivisionError 등) |
| `WA` | Wrong Answer | 로직 오류로 인한 오답 |
| `TLE` | Time Limit Exceeded | 시간 초과, 알고리즘 최적화 필요 |
| `MLE` | Memory Limit Exceeded | 메모리 초과 |
