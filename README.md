## 🎯 Platforms

### [백준 (Baekjoon Online Judge)](https://www.acmicpc.net/)
- 한국의 대표적인 온라인 저지 플랫폼
- 다양한 난이도의 알고리즘 문제 제공
- 각 문제는 문제 번호로 폴더 구성

### [프로그래머스 (Programmers)](https://programmers.co.kr/)
- 코딩테스트 대비 플랫폼
- Level 1~5로 난이도 구분
- 실전 코딩테스트 형식의 문제 제공

## 📝 How to Use

1. 각 플랫폼 폴더에서 문제를 선택합니다
2. 문제 폴더 내 README.md에서 문제 설명과 접근 방법을 확인합니다
3. solution 파일에서 코드 구현을 확인합니다
