# 백준 (Baekjoon Online Judge)

백준 온라인 저지에서 해결한 문제들을 정리합니다.

## 📁 폴더 구조

각 문제는 문제 번호로 폴더를 만들어 관리합니다.

```
Baekjoon/
├── 1000/
│   ├── solution.py
│   └── README.md
├── 2000/
│   ├── solution.cpp
│   └── README.md
```

## 📝 문제 정리 형식

각 문제 폴더에는 다음을 포함합니다:
- **solution.{ext}**: 문제 풀이 코드
- **README.md**: 문제 설명, 접근 방법, 시간복잡도 등

### README.md 예시

```markdown
# [문제 번호] 문제 제목

## 문제 링크
https://www.acmicpc.net/problem/{문제번호}

## 문제 설명
(문제 요약)

## 접근 방법
(문제 해결 아이디어)

## 시간 복잡도
O(n)

## 사용 언어
Python / C++ / Java
```
